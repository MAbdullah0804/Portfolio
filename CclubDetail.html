Project Overview

Gaming Club is a sleek, interactive web platform designed to bring gaming enthusiasts together in a vibrant online community. Built with HTML, CSS (Bootstrap-enhanced), and JavaScript, the site features:

  • Hero Section: A visually striking introduction with call-to-action for new members.
  • Featured Games: Interactive card-based layout showcasing top games like Tic Tac Toe and Rock Paper Scissors, with direct links to play.
  • Join Section: Simple, user-friendly sign-up form to engage users and grow the community.
  • Responsive Design: Optimized for mobile and desktop, ensuring seamless gameplay and browsing.
  • Modern UI: Smooth hover effects, bold colors, and subtle animations to create a premium gaming vibe.
