Cafe Website

A modern, responsive frontend web application showcasing real-world UI design and JavaScript interactivity.

HTML5 CSS3 JavaScript Bootstrap 5

Project Overview

This project is a responsive cafe website designed to simulate a real-world ordering experience. Users can browse menu items, add them to a cart, and dynamically view order totals. The project focuses on clean UI, responsiveness, and interactive frontend logic.

Key Features

  • Responsive layout using Bootstrap 5 grid system
  • Interactive cart and order handling
  • Dynamic price calculation
  • Clean and modern UI design
  • Mobile-first and cross-browser friendly

Technologies & Concepts

HTML5

Semantic structure, accessibility, clean layout.

CSS3

Custom styling, gradients, shadows, UI polish.

Bootstrap 5

Responsive grid system and UI components.

JavaScript

DOM manipulation, events, dynamic updates.
