Universal DASH / HLS / MP4 Player

A modern, responsive web-based media player supporting adaptive streaming and local video playback with a clean, scalable UI.

Project Overview

Universal DASH Player is a browser-based multimedia application engineered to support adaptive streaming technologies such as MPEG-DASH and HLS, alongside standard video formats including MP4, WebM, and MKV. The project demonstrates strong frontend architecture, efficient media handling, and a focus on user experience.

Key Features

Adaptive streaming (DASH & HLS)
Dynamic quality switching
Cross-browser compatibility
Local video playback support
Responsive & mobile-first UI
Robust error & event handling

Technologies & Concepts

HTML5

Semantic markup, accessibility-focused structure, and native video elements for scalable media playback.

CSS3

Responsive layouts, modern UI effects, gradients, glassmorphism, and visual hierarchy.

JavaScript

DOM manipulation, event handling, asynchronous media loading, and adaptive streaming logic.

Bootstrap 5

Responsive grid system, reusable UI components, and mobile-first design principles.

How It Works

Users can load either an online streaming source or a local video file. JavaScript dynamically initializes playback by attaching the media source to the HTML5 video element. Adaptive streaming libraries handle manifest parsing, buffering, and quality switching based on real-time network conditions.
